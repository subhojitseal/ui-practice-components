#ui-practice components
A collection of custom UI components, creative buttons, and smooth CSS animations I've built while sharpening my front-end skills.

[🚀 View Live Demo](https://github.io)

---

## 🛠️ What's Inside?

### 🖱️ Interactive Buttons
* **Neon Glow:** Hover effects using CSS shadows.
* **3D Press:** Depth-based animations.
* **Loading States:** Transitioning from text to spinners.

### ✨ Animations & UI
* **Glassmorphism Cards:** Modern UI with backdrop filters.
* **Smooth Transitions:** Easing functions for menu toggles.
* **SVG Animations:** Keyframe-based path movements.
