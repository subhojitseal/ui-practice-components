#ui-practice components

A showcase of modern web design techniques focusing on animations, depth, and interactive lighting effects. This project contains three distinct UI experiments built with pure CSS.

---

## 🔥 Featured Effects

### 1. Neon Animation Button
*   **Concept:** High-contrast glow effect using `box-shadow` and `text-shadow`.
*   **Highlights:** Smooth hover transitions and pulsing border-light animations.

### 2. 3D Flip Card Effect
*   **Concept:** Utilizing `perspective` and `transform-style: preserve-3d`.
*   **Highlights:** A seamless 180-degree flip to reveal back-side content on hover.

### 3. Interactive Card Hover
*   **Concept:** Elevation and depth changes using CSS `transform` and `transition`.
*   **Highlights:** Modern "pop-out" effects and smooth scaling for a tactile feel.

---

## 🛠️ Built With
*   **HTML5:** Clean, semantic markup.
*   **CSS3:** Keyframes, Flexbox, and 3D Transforms.
